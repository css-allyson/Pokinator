# Pokinator

Agente inteligente que adivinha Pokémons através de perguntas, utilizando os classificadores Árvore de Decisão e Naive Bayes.

## Instalação

### Pré-requisitos

- Python 3.11+
- [uv](https://docs.astral.sh/uv/)

### Configuração

```bash
# Instalar dependências
uv sync

# Instalar dependências de desenvolvimento
uv sync --group dev
```

## Uso

```bash
# Iniciar a interface web
uv run task app
```
